Python语言基础知识之输入输出 

通常，任何程序都会通过输入和输出功能来实现与用户进行交互和沟通。所谓输入，就是指程序通过用户输入的数据获取信息；而输出则是指程序向用户显示或打印数据。

一、数据的输入

1. 用input()函数输入数据

input()函数可以输入数值、字符串和其他对象。执行时首先在屏幕上显示提示字符串，然后等待用户输入，输入完毕后按回车，并将用户输入作为一个表达式进行解释、求值，最后将结果赋给指定变量。


input()函数也可以直接用在表达式中，相当于一个值。


2. 用raw_input()函数输入数据

input()函数在输入数值型数据时相对比较方便，但是在输入字符串时则需要加上引号，否则input会将输入的字符串解释为变量名，除非程序中定义过该变量，否则将会报“变量未定义”的错误。
